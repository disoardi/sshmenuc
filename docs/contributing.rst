Contributing
============

Please see the `CONTRIBUTING.md <https://github.com/disoardi/sshmenuc/blob/main/CONTRIBUTING.md>`_ file in the repository for detailed contribution guidelines.

Development Setup
-----------------

1. Clone repository
2. Install Poetry
3. Run ``poetry install``
4. Activate environment: ``poetry shell``

Running Tests
-------------

::

    make test          # Run all tests
    make coverage      # Generate coverage report
    make lint          # Check code style

--------------------

5. Submit pull request
