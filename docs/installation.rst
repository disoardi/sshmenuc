Installation
============

Requirements
------------

- Python 3.9 or higher
- Poetry (for development)
- tmux (optional, for session management)

Install from PyPI
-----------------

::

    pip install sshmenuc

Install from Source
-------------------

::

    git clone https://github.com/disoardi/sshmenuc.git
    cd sshmenuc
    poetry install
    poetry shell
